STATIC WEB SITE BOOTSTRAP
=========================

A set of PHP/JS/CSS/HTML files that help you set up a static web site in no time.

What it does
------------

Gives an idea of a structure of your new static HTML site.

Dependencies
------------

PHP 5.2 / MySQL 5 / jQuery
